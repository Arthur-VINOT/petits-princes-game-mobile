Cahier des Charges - La Course des Petits Princes



1. Présentation du projet

"La Course des Petits Princes" est un jeu web 2D de type runner, développé pour sensibiliser aux actions de l'association Les Petits Princes qui réalise les rêves d'enfants malades. Le joueur contrôle un personnage qui court et saute pour collecter des pièces et des billets, symbolisant les dons permettant de réaliser ces rêves.

2. Objectifs du jeu

Objectif principal : Collecter suffisamment de "dons" (pièces et billets) pour réaliser le rêve d'un enfant.
Message social : Sensibiliser les joueurs à l'importance des dons pour l'association Les Petits Princes.
Expérience ludique : Proposer un jeu simple et amusant accessible à tous.

3. Spécifications techniques

Environnement de développement

Langage : HTML5, CSS3, JavaScript
Support : Web (navigateurs modernes)
Responsive : Adaptation à différentes tailles d'écran

Architecture technique

Canvas HTML5 pour le rendu graphique
LocalStorage pour la gestion des comptes utilisateurs
Audio HTML5 pour la musique de fond

4. Fonctionnalités
Système d'authentification

Création de compte (nom d'utilisateur/mot de passe)
Connexion utilisateur
Sauvegarde locale des informations de compte

Interface utilisateur

Écran de connexion
Écran d'accueil avec instructions
Interface de jeu
Écran de fin avec lien vers le site de donation
Messages contextuels (niveau supérieur, collision)

Gameplay

Contrôles : Touches du clavier (espace pour sauter)
Mécaniques :

Gravité et saut
Obstacles à éviter
Collecte de pièces (1€) et billets (10€)
Difficulté progressive (vitesse augmentant avec le temps)
Système de niveaux basé sur le score

Éléments de jeu

Personnage jouable : Le "prince"
Collectibles : Pièces (or) et billets (verts)
Obstacles : Blocs rouges
Environnement : Fond fixe avec défilement des éléments

5. Règles du jeu

Le joueur commence avec 0€ de dons
Le but est d'atteindre un montant cible pour gagner
Chaque pièce collectée rapporte 1€
Chaque billet collecté rapporte 10€
Les obstacles ralentissent la progression
La vitesse du jeu augmente progressivement
À chaque palier de 10€, le joueur monte d'un niveau

6. Design graphique et sonore

Éléments visuels

Personnage : Sprite "prince.png"

Éléments sonores

Musique de fond : Fichier "music.mp3" en boucle

7. Flux utilisateur

L'utilisateur arrive sur l'écran de connexion
Après connexion/inscription, il accède à l'écran d'accueil
Il démarre le jeu via le bouton "Commencer"
Il joue jusqu'à atteindre l'objectif ou quitter
À la fin, il peut faire un don réel, rejouer ou se déconnecter
